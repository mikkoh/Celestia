# Celestia
3D Einsteinian gravity simulator.

### Development
Install [Node.js].

Then, from the root, run

```sh
> npm install
```

To then watch for any file updates, run

```sh
> gulp watch
```

In a second terminal, run

```sh
> npm start
```
